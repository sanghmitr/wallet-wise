Wallet Wise MVP Requirements

Overview
- Monorepo workspace with `client` and `server`
- Frontend: React + Vite + Tailwind CSS + TypeScript
- Backend: Express + TypeScript
- Database: Firebase Firestore with memory fallback
- AI: Gemini API
- Charts: Recharts
- PWA: `vite-plugin-pwa`

Environment
- Node.js 20+
- npm 10+
- Firebase project with Firestore enabled
- Firebase Authentication enabled for Google and/or Anonymous sign-in
- Gemini API key

Root Workspace Install
- `npm install`

Root Workspace Packages
- Workspace management is handled from the root `package.json`
- Root scripts:
- `npm run dev:client`
- `npm run dev:server`
- `npm run build`
